# My Mac provisioning

http://t-wada.hatenablog.jp/entry/mac-provisioning-by-ansible

これをもとに、自分用に色々手を加えたものです。

## Post provisioning

- iTerm の default の実行コマンドを `/usr/local/bin/zsh` に
- 秘伝の `~/.ssh` を持ってくる
- 秘伝の `~/.zshrc` を持ってくる
- Dropbox 同期後、`~/Dropbox/dotfiles` で `rake dot:install`
